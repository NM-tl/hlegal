# Install

To run this project locally, follow these steps:

##Clone the repository:

   ```bash
   git clone https://github.com/NM-tl/hlegal.git

1. Navigate to the project directory:

cd hlegal

2. Install dependencies:

# If using npm
npm install

# If using yarn
yarn

3. Run the development server:

# If using npm
npm run dev

# If using yarn
yarn dev
```
